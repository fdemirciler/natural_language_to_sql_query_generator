SQL Query Generator

Convert natural language questions into SQL queries using AI

Project Overview

This application converts natural language questions into SQL queries for a world cities database using Together.ai's language models. Users can ask questions in plain English, and the application will generate the corresponding SQL query, execute it against a Supabase database, and display the results.

Try the Live App

Features

  • Natural language to SQL conversion using Together.ai
  • Real-time query execution against a Supabase database
  • Syntax highlighting for SQL queries
  • Clean, responsive user interface built with Chakra UI
  • Error handling and loading states

Example Questions

Try asking questions like:

  • "How many cities are there in each country?"
  • "What are the top 10 most populated cities?"
  • "Which countries have the highest life expectancy?"
  • "List all cities in Germany with population over 500,000"

Technical Implementation

This project uses:

  • Next.js for the frontend and API routes
  • Vercel AI SDK for secure Together.ai integration
  • Chakra UI for a clean, responsive interface
  • Supabase for database operations
  • Vercel for deployment and serverless functions

How It Works

When a user submits a question:

  1. The application sends the question and database schema to the Together.ai API
  2. Together.ai generates a SQL query based on the question and schema
  3. The application executes the query against the Supabase database
  4. Results are displayed in a clean, tabular format
